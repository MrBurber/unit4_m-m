What is the 8051?

The Intel 8051 is an 8-bit single-chip microcontroller introduced in 1980. It integrates CPU, RAM, ROM, I/O ports, timers, and serial communication on a single chip — a complete embedded system solution.

Programmer's Model of 8051
  • Register Banks (0–3): Four banks of 8 registers (R0–R7). Selected by RS1:RS0 bits in PSW. Bank 0 at 00H–07H.
  • Bit-Addressable Area: RAM locations 20H–2FH — 128 individually addressable bits.
  • General Purpose RAM: Scratch-pad area 30H–7FH.
  • Stack: Grows upward from SP; defaults inside internal RAM (max 128 bytes).
PSW Flags: CY (Carry) · AC (Auxiliary Carry) · F0 (User Flag) · RS1:RS0 (Register Bank Select) · OV (Overflow) · P (Parity)

Memory Interfacing & Control Signals
  • ALE (Address Latch Enable): Pulses at Fosc/6. 74LS373 latch demultiplexes P0 into low address byte (A0–A7).
  • PSEN̄ (Program Store Enable): Goes low during external ROM fetch to enable ROM output.
  • WR̄ / RD̄: P3.6 / P3.7 — used for external RAM write/read with MOVX instruction.
  • External ROM Access: PSEN̄ activates, P2 gives high address, P0 gives low address then data.
  • External RAM Access: MOVX @DPTR,A or MOVX A,@DPTR — WR̄/RD̄ activates on P3.

Sample Program — LED Blink on P1.0
ORG  0000H
START:
    CPL   P1.0       ; Toggle LED on P1.0
    ACALL DELAY      ; Wait
    SJMP  START      ; Loop forever

DELAY:
    MOV   R0, #0FFH  ; Outer count = 255
D1: MOV   R1, #0FFH  ; Inner count = 255
D2: DJNZ  R1, D2    ; Inner loop
    DJNZ  R0, D1    ; Outer loop
    RET              ; Return to caller

END

Timer / Counter Programming

8051 has Timer 0 (T0) and Timer 1 (T1) — both 16-bit, configured via TMOD register (not bit-addressable).

TMOD Register — Timer Modes

M1:M0ModeDescription
00Mode 013-bit timer/counter (for 8048 compatibility)
01Mode 116-bit timer/counter — most commonly used
10Mode 28-bit auto-reload (TH loaded into TL on overflow)
11Mode 3Split: T0 = two independent 8-bit timers; T1 = frozen
Timer vs Counter: C/T̄ bit in TMOD. C/T=0 → Timer (counts internal machine cycles = Fosc/12). C/T=1 → Counter (counts falling edges on T0/T1 pin).
Delay Formula (Mode 1, 12 MHz): Machine cycle = 1 μs. Delay = (65536 – TH:TL initial value) × 1 μs.
Example: For 50 ms delay → TH:TL = 65536 – 50000 = 15536 = 3CB0H
Interrupt System

8051 has 6 interrupt sources with 2 priority levels (high/low) configurable via IP register.

InterruptVector AddressSource
Reset0000HHardware reset (not a true interrupt)
INT0̄ (EX0)0003HExternal interrupt 0 — P3.2 (level or edge)
TF0 (ET0)000BHTimer 0 overflow flag
INT1̄ (EX1)0013HExternal interrupt 1 — P3.3 (level or edge)
TF1 (ET1)001BHTimer 1 overflow flag
RI/TI (ES)0023HSerial port — receive complete or transmit empty

IE Register (A8H) — Interrupt Enable

EA Global enable (bit 7). Individual enables: EX0 ET0 EX1 ET1 ES

Default Priority: INT0 > TF0 > INT1 > TF1 > Serial. Two levels: set bits in IP register for high priority. ISR must end with RETI (not RET) to correctly restore interrupt system.
IT0/IT1 bits in TCON: 0 = level triggered (low level), 1 = edge triggered (falling edge) for INT0̄/INT1̄.
Serial Communication (UART)

Full-duplex serial port using RXD (P3.0) and TXD (P3.1). Timer 1 in Mode 2 (auto-reload) is the standard baud rate generator.

SCON ModeSM0:SM1Description
Mode 000Shift register I/O through RXD; fixed Fosc/12
Mode 1018-bit UART (start + 8 data + stop); variable baud
Mode 2109-bit UART; fixed baud Fosc/32 or Fosc/64
Mode 3119-bit UART; variable baud (like Mode 1 + 9th bit)
Baud Rate (Mode 1/3): Baud = Fosc ÷ (384 × (256 – TH1))
Common: 9600 baud @ 11.0592 MHz crystal → TH1 = FDH
SCON bits: SM0:SM1 (mode) · SM2 (multiprocessor) · REN (receiver enable) · TB8/RB8 (9th bit) · TI (transmit interrupt) · RI (receive interrupt)
